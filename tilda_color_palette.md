# Палитра цветов сайта Tilda

## Основные цвета (извлечены с http://zapisvmdsa.tilda.ws/page111876146.html)

### Основной акцентный цвет (кнопки, CTA)
- **RGB:** `rgb(59, 70, 238)`
- **HEX:** `#3B46EE`
- **Использование:** Кнопка "Начать тест", основные акценты

### Вторичный акцент (кнопки действий)
- **RGB:** `rgb(0, 0, 0)`
- **HEX:** `#000000`
- **Использование:** Кнопка "Связаться", вторичные действия

### Текст
- **Основной текст:** `rgb(0, 0, 0)` / `#000000`
- **Фон:** Белый / прозрачный

### Дополнительные цвета (рекомендуемые для градиентов)
- **Светлый оттенок основного:** `#5B66F0` (для hover эффектов)
- **Темный оттенок основного:** `#2B36CC` (для активных состояний)

## Применение в Streamlit

Эти цвета будут использованы для:
- Кнопок и CTA элементов
- Акцентов и выделений
- Градиентов метрик
- Hover эффектов
- Прогресс-баров
